# Задание 2: Параметрический тест с `pytest`

Этот проект содержит функцию `decode` и параметрические тесты для нее.

## Установка зависимостей

1.  Установите `pytest`:
    ```bash
    pip install pytest
    ```

## Запуск тестов

1.  Выполните в терминале следующую команду:
    ```bash
    pytest
    ```

## Проверка стиля кода (flake8)

1.  Установите `flake8`, если он не установлен: `pip install flake8`
2.  Запустите проверку:
    ```bash
    flake8 morse.py test_morse.py
    ```
    Ожидается, что команда не вернет никакого вывода.
