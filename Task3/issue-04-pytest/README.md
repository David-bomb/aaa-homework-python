# Задание 4: Тестирование с `pytest`

Проект содержит функцию `fit_transform` и набор тестов для нее с использованием фреймворка `pytest`.

## Установка зависимостей

1.  Установите `pytest`: `pip install pytest`

## Запуск тестов

1.  Выполните в терминале следующую команду:
    ```bash
    pytest
    ```

## Проверка стиля кода (flake8)

1.  Установите `flake8`: `pip install flake8`
2.  Запустите проверку:
    ```bash
    flake8 one_hot_encoder.py test_one_hot_encoder_pytest.py
    ```
