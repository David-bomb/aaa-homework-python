# Задание 1: Тестирование с `doctest`

Этот модуль содержит функцию `encode` для кодирования строки в азбуку Морзе и встроенные тесты `doctest`.

## Запуск тестов

1.  Убедитесь, что у вас установлен Python.
2.  Выполните в терминале следующую команду, находясь в директории с файлом:

    ```bash
    python -m doctest -v morse_doctest.py
    ```

## Проверка стиля кода (flake8)

1.  Установите `flake8`, если он не установлен: `pip install flake8`
2.  Запустите проверку:
    ```bash
    flake8 morse_doctest.py
    ```
    Ожидается, что команда не вернет никакого вывода, что означает отсутствие ошибок.
